–ê –≤–æ—Ç –¥–ª—è –º–æ–±–∏–ª—å–Ω—ã—Ö –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π —Ç–æ–∫–µ–Ω—ã —É–¥–æ–±–Ω–µ–µ –≤—Å–µ–≥–æ –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å —á–µ—Ä–µ–∑ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ HTTP-–∑–∞–≥–æ–ª–æ–≤–∫–∏. –¢–∞–º —É–∂–µ –Ω–µ —Ç–∞–∫ –ø—Ä–æ—Å—Ç–æ –∑–ª–æ—É–º—ã—à–ª–µ–Ω–Ω–∏–∫—É —Å—á–∏—Ç–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –∏–∑ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è. –ù–æ –≤–æ—Ç —Ä–∞–±–æ—Ç–∞ —Å –∫—É–∫–∞–º–∏ —ç—Ç–æ —É–∂–µ –æ—Ç–¥–µ–ª—å–Ω—ã–π –∞–¥.
